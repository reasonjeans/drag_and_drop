# Drag and Drop

react-beautiful-dnd 라이브러리를 사용한 드래그 앤 드롭 프로젝트

https://reasonjeans.github.io/drag_and_drop


## using
- Typescript
- react-beautiful-dnd
- Recoil
- Styled Components


## 기능
- react-beautiful-dnd 라이브러리를 이용한 드개르 앤 드롭 구현
- form 리스트 작성 구현



## 설치 및 실행
설치
```
$ npm i
```

실행
```
$ npm start
```
